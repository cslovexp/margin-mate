# Política de privacidad de Calculadora de margen

Fecha de entrada en vigor: 19 de mayo de 2026

Calculadora de margen (la "Aplicación") es una herramienta de cálculo que ayuda a pequeños negocios y vendedores en línea a calcular precios de venta, costes, comisiones, gastos de envío, IVA, descuentos, beneficio y margen.

Esta Política de privacidad explica qué información procesa la Aplicación y cómo se almacenan y gestionan los datos del usuario.

## 1. Recogida de información personal

La Aplicación no requiere registro de cuenta y no recoge información personal que identifique directamente a los usuarios, como nombre, número de teléfono, dirección de correo electrónico, información de ubicación, contactos o fotos.

La Aplicación no utiliza SDK publicitarios ni SDK de inicio de sesión social, y no transmite los datos de cálculo de los usuarios a un servidor operativo. Sin embargo, puede usar Google Analytics for Firebase para mejorar la calidad del servicio, analizar errores y comprender el uso de las funciones.

## 2. Datos procesados en la Aplicación

La Aplicación puede procesar la siguiente información introducida directamente por el usuario en el dispositivo para proporcionar funciones de cálculo y comodidad.

- Nombre del producto o título del cálculo
- Precio de venta, coste, tasa de comisión, tasa de comisión de pago, gastos de envío, costes de embalaje, costes de publicidad
- Margen objetivo, tasa de descuento y valores relacionados con el IVA
- Resultados de cálculo, historial de cálculo guardado, plantillas y ajustes de la aplicación

Esta información son datos de cálculo comercial introducidos directamente por el usuario para utilizar las funciones de la Aplicación. El operador de la Aplicación no recoge ni transmite estos datos por separado a un servidor.

## 3. Almacenamiento de datos

El historial de cálculo, las plantillas y los ajustes predeterminados se almacenan en el almacenamiento local del dispositivo del usuario.

Los datos almacenados pueden eliminarse cuando se elimina la Aplicación, se restablece el dispositivo o el usuario elimina registros en la Aplicación.

## 4. Copia de seguridad, exportación y uso compartido

Si el usuario lo elige, la Aplicación puede realizar copias de seguridad del historial de cálculo como archivos o compartir resultados de cálculo en PDF, imagen, texto o formatos similares.

En este caso, los datos pueden entregarse a la aplicación externa, ubicación de almacenamiento o destinatario de uso compartido seleccionado por el usuario. La Aplicación solo envía dichos datos externamente cuando el usuario realiza directamente una acción de compartir o exportar.

## 5. Uso compartido con terceros

La Aplicación no vende, proporciona ni comparte información personal o datos de cálculo de los usuarios con terceros.

Sin embargo, si el usuario selecciona directamente funciones de Android o iOS para compartir, guardar archivos o imprimir, pueden aplicarse las políticas del sistema operativo o de las aplicaciones externas que proporcionan dichas funciones.

## 6. Permisos

La Aplicación no requiere permisos sensibles del dispositivo para las funciones de cálculo en sí.

Al utilizar determinadas funciones, como la selección de archivos de copia de seguridad, el guardado de archivos, el uso compartido o la impresión, puede iniciarse el selector de archivos, la hoja de compartir o la función de impresión del sistema operativo. Este proceso se realiza únicamente mediante una acción explícita del usuario.

## 7. Privacidad de los menores

La Aplicación no está dirigida a menores de 14 años y no recoge conscientemente información personal de menores.

## 8. Conservación y eliminación de datos

El historial de cálculo y los ajustes almacenados en la Aplicación se conservan en el dispositivo del usuario.

Los usuarios pueden eliminar los datos almacenados usando las funciones de eliminación de la Aplicación o eliminando la Aplicación. Los archivos guardados externamente mediante copia de seguridad o uso compartido deben ser eliminados directamente por el usuario desde la ubicación de almacenamiento o aplicación externa correspondiente.

## 9. Análisis de uso de la aplicación

La Aplicación puede usar Google Analytics for Firebase para comprender el uso de pantallas y funciones. Los datos de cálculo ingresados por el usuario, como precios, costos, márgenes, nombres de productos, categorías, notas o historiales detallados, no se envían como eventos de análisis.

Los identificadores publicitarios y las señales de personalización de anuncios están desactivados. Los eventos de análisis se usan únicamente para mejorar el producto, verificar errores y comprender el uso general de las funciones.

## 10. Cambios en esta Política de privacidad

Esta Política de privacidad puede actualizarse debido a cambios en las funciones de la Aplicación, leyes aplicables o políticas de las tiendas.

Si hay cambios importantes, se podrá proporcionar un aviso mediante notificaciones dentro de la Aplicación o páginas de distribución.
