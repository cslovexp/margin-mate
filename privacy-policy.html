마진 계산기 개인정보처리방침

시행일: 2026년 5월 19일

마진 계산기는 회원가입 없이 사용할 수 있는 손익/마진 계산 도구이며, 사용자의 계산 데이터를 운영 서버로 전송하지 않습니다.

마진 계산기(이하 "앱")는 소상공인과 온라인 판매자가 판매가, 원가, 수수료, 배송비, 부가세, 할인, 손익 및 마진율을 계산할 수 있도록 돕는 계산 도구입니다. 본 개인정보처리방침은 앱이 어떤 정보를 처리하고, 사용자의 데이터가 어떻게 저장 및 관리되는지 설명합니다.

1. 개인정보 수집 여부

앱은 회원가입을 요구하지 않으며, 이름, 전화번호, 이메일 주소, 위치정보, 연락처, 사진 등 사용자를 직접 식별할 수 있는 개인정보를 수집하지 않습니다.

앱은 광고 SDK, 분석 SDK, 소셜 로그인 SDK를 사용하지 않으며, 사용자의 계산 데이터를 운영 서버로 전송하지 않습니다.

2. 앱에서 처리되는 데이터

앱은 사용자가 직접 입력한 다음 정보를 계산과 편의 기능 제공을 위해 기기 내에서 처리할 수 있습니다.

  • 상품명 또는 계산 제목
  • 판매가, 원가, 수수료율, 결제 수수료율, 배송비, 포장비, 광고비
  • 목표 마진율, 할인율, 부가세 관련 입력값
  • 계산 결과, 저장된 계산 기록, 템플릿, 앱 설정값

이 정보는 사용자가 앱 기능을 이용하기 위해 직접 입력한 업무용 계산 데이터이며, 앱 운영자가 별도로 수집하거나 서버로 전송하지 않습니다.

3. 데이터 저장 방식

계산 기록, 템플릿, 기본 설정값은 사용자의 기기 내 로컬 저장소에 저장됩니다. 앱 삭제, 기기 초기화, 사용자의 기록 삭제 조작이 있는 경우 저장된 데이터가 삭제될 수 있습니다.

4. 백업, 내보내기, 공유 기능

앱은 사용자가 선택한 경우 계산 기록을 파일로 백업하거나, 계산 결과를 PDF, 이미지, 텍스트 등의 형태로 공유할 수 있습니다.

이 경우 데이터는 사용자가 선택한 외부 앱, 저장 위치, 공유 대상에 전달될 수 있습니다. 앱은 사용자가 공유 또는 내보내기를 직접 실행한 경우에만 해당 데이터를 외부로 전달합니다.

5. 제3자 제공

앱은 사용자의 개인정보 또는 계산 데이터를 제3자에게 판매, 제공, 공유하지 않습니다.

다만 사용자가 Android 또는 iOS의 공유 기능, 파일 저장 기능, 인쇄 기능을 직접 선택하는 경우 해당 기능을 제공하는 운영체제 또는 외부 앱의 정책이 적용될 수 있습니다.

6. 접근 권한

앱은 계산 기능 자체를 위해 민감한 기기 권한을 요구하지 않습니다.

백업 파일 선택, 파일 저장, 공유, 인쇄 등 일부 기능을 사용할 때 운영체제의 파일 선택기, 공유 시트, 인쇄 기능이 실행될 수 있습니다. 이 과정은 사용자의 명시적인 조작에 의해 수행됩니다.

7. 아동의 개인정보

앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 고의로 수집하지 않습니다.

8. 데이터 보관 및 삭제

앱에 저장된 계산 기록과 설정값은 사용자의 기기 내에 보관됩니다.

사용자는 앱 내 삭제 기능을 사용하거나 앱을 삭제하여 저장된 데이터를 삭제할 수 있습니다. 백업 또는 공유를 통해 외부에 저장된 파일은 사용자가 해당 저장 위치 또는 외부 앱에서 직접 삭제해야 합니다.

9. 개인정보처리방침 변경

앱 기능 변경, 관련 법령 변경, 스토어 정책 변경에 따라 본 개인정보처리방침이 수정될 수 있습니다. 중요한 변경 사항이 있는 경우 앱 내 공지 또는 배포 페이지를 통해 안내할 수 있습니다.
